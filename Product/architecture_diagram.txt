CORE FLOWS:
1. User Registration & Scene Creation:
   User → API Gateway → User Service → Scene Service → PostgreSQL

2. Daily Blog Generation:
   Scheduled Job → Content Service → Location Service → Scene Service → Blog Generation Service → LLM API → PostgreSQL/MongoDB → Notification Service → User

3. Content Discovery:
   External APIs → Content Service → MongoDB → Elasticsearch → Blog Generation Service

4. User Interaction:
   User → API Gateway → Content Service → Analytics Service → Scene Service (for preference updates)
